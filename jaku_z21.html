Co znaczy skrót ID? Identyfikator – unikalna nazwa nadawana elementowi HTML.
Jaki jest cel stosowania ID? Umożliwia jednoznaczne wskazanie elementu do stylowania CSS lub działania JavaScript.
Jakie jest znaczenie znaku # W CSS znak # służy do odwoływania się do elementu z konkretnym ID.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Poprzez zapis: #nazwaID w CSS, np. #naglowek {color: red;}
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia. 1. ID musi być unikalne w całym dokumencie.
2. Zaczyna się od litery, nie może zawierać spacji.
3. Może być używane do nawigacji (kotwice) oraz manipulacji przez JavaScript.
